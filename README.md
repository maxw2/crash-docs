## Crash-docs （组件开发的文档工具）
创建一个最基础的文档工具 熟悉webpack使用

### v1.0版本 开发功能
- [ ] 1. config.js 配置项
  - [ ] 1.1 首页的标题 内容的配置 
  - [ ] 1.2 菜单列表根据标题 自动生成路由
  - [ ] 1.3 可以引入图片
- [ ] 2. 详情页
  - [ ] 2.1 详情页可以使用Markdown编写
  - [ ] 2.2 React / Vue 所编写的组件不用进行注册 直接写入Markdown
  - [ ] 2.3 支持Ts编写组件
- [ ] 3. 页面布局 
  - [ ] 3.1 自适应多端 
  - [ ] 3.2 双栏分布 
  
  参考链接：
  [dumi](https://d.umijs.org/zh-CN)
